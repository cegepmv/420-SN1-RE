* **Python est le langage n°1 en 2026** selon l'indice TIOBE, devançant tous ses concurrents grâce à sa polyvalence.
* **Python est multiplateforme :** Il fonctionne parfaitement sur Windows, macOS, Linux, et même sur des microcontrôleurs (comme le Raspberry Pi ou l'ESP32).
* **Une syntaxe ultra-simple :** Contrairement à Java ou C++, Python se lit presque comme de l'anglais. C'est le langage idéal pour débuter car il nécessite moins de lignes de code pour accomplir la même tâche.
* **Polyvalence extrême :** Que vous fassiez de l'intelligence artificielle, du web ou de la finance, Python possède une bibliothèque pour tout.
* **Domaines d'utilisation :**
* **Intelligence Artificielle & Data Science :** Le leader absolu (TensorFlow, PyTorch, Pandas).
* **Développement Web :** Création de sites robustes avec Django ou Flask.
* **Automatisation :** Idéal pour créer des scripts qui automatisent des tâches répétitives.
* **Calcul scientifique :** Utilisé par les chercheurs et ingénieurs du monde entier.
* **Cybersécurité :** Outil de prédilection pour le hacking éthique et l'analyse de réseaux.

## Qu'est-ce que Python ?

**Python** est un langage de programmation **interprété**, **orienté objet** et de **haut niveau**. Sa philosophie repose sur la lisibilité du code. Contrairement à Java qui doit être compilé en "bytecode", Python est exécuté ligne par ligne par un interpréteur, ce qui rend le cycle de développement et de test beaucoup plus rapide pour le développeur.

# Installer Python

<div style="color:red">POUR LINUX (Ubuntu/Debian)</div>

Python est souvent préinstallé. Pour installer la dernière version :

```bash
sudo apt update
sudo apt install python3 python3-pip

```

---

<div style="color:red">POUR WINDOWS</div>

1. Allez sur le site officiel [python.org](https://www.python.org/downloads/).
2. Téléchargez l'installeur (ex: Python 3.12+).
3. **IMPORTANT :** Cochez la case **"Add Python to PATH"** lors de l'installation pour pouvoir l'utiliser dans votre terminal.

---

<div style="color:red">POUR MACOS</div>

Utilisez le gestionnaire de paquets Homebrew pour une gestion simplifiée :

```bash
brew install python

```
